# bp-indicator-nav

Tableau navigation extensions for the BP Diagnostic Tool.

## Extensions

| File | Purpose |
|---|---|
| `overview_nav.html` | Subcategory card navigation — overview → indicator table |
| `indicator_nav.html` | Indicator row navigation — indicator table → child dashboard |

## Setup

1. Clone repo
2. Copy `tableau.extensions.js` from `b_tableau_extensions_2`
3. Push to GitHub, enable Pages on `main` branch / root
4. Load `.trex` files into Tableau dashboards via **Extension object**

See `CLAUDE.md` for full field and parameter configuration.
